# 💼 Job Trackkrr

Job Trackkrr is a modern job application tracking system built with React, TypeScript, and Vite.  
It helps users manage, organize, and monitor their job applications in one clean dashboard.

---

## 🚀 Features

- Add new job applications
- Track job status (Applied, Interview, Offer, Rejected)
- Filter and search jobs
- Responsive UI (mobile-friendly)
- Authentication with JWT (cookie-based)
- Persistent data storage
- Clean modern UI with Tailwind CSS

---

## 🛠 Tech Stack

### Frontend

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router

---

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/job-trackkrr.git
cd job-trackkrr
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start development server

```bash
npm run dev
```

App runs at:

```
http://localhost:4000
```
